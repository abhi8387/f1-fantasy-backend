# F1 Fantasy Backend

This is the backend for the F1 Fantasy project, built with Express.js, Sequelize, and MySQL.

## Features

- User authentication (JWT, bcrypt)
- Database management with Sequelize and MySQL
- RESTful API with Express.js
- Real-time data fetching with Axios

## Dependencies

- `express`
- `sequelize`
- `mysql2`
- `jsonwebtoken`
- `bcrypt`
- `axios`
- `dotenv`
- `nodemon` (dev)

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your environment variables in a `.env` file (e.g., `DB_HOST`, `DB_USER`, `DB_PASS`, `DB_NAME`, `JWT_SECRET`).
4. Run the backend: `npm run dev`
